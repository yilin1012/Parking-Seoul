# Parking-Seoul

## 1. 프로젝트 소개 :

본 프로젝트의 제목은 <파킹서울(Parking Seoul)>이다. <파킹서울>은 서울시 공영주차장 안내 및 결제 프로그램이 다. <파킹서울>의 목표는 파이썬 GUI를 기반으로 서울시 공영 주차장 정보를 제공하고, 프로그램을 통해 주차 요금을 정산 하도록 하는 것이다.

## 2. 사용 데이터 :
서울열린데이터광장 제공 서울시 공영주차장 안내 정보

## 3. 개발 언어 :
python

## 4. 사용 라이브러리 :
pandas, tkinter, time, datetime
